LoboBrowser: Java Web Browser

LoboBrowser is an open source web browser that is written completely in Java. LoboBrowser is being actively developed with the aim to fully support HTML 5, Javascript and CSS3. LoboBrowser also supports direct JavaFX rendering. The general goal of the LoboBrowser effort is to produce a browser that is fast, complete, easy to extend, feature-rich and secure.
